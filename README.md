# Reparadora de Calzados y Ortopedia - E-commerce

Sistema de ventas web para cadena de reparadora de calzados. Proyecto con doble propósito: negocio real y aprendizaje de desarrollo FullStack.

## 🛠️ Stack Tecnológico

- **Frontend:** Next.js, React, Tailwind CSS, TypeScript
- **Base de Datos:** Supabase (PostgreSQL)
- **Control de Versiones:** Git + GitHub

## 🎯 Objetivo

Desarrollar un e-commerce funcional que permita:
- Catálogo de productos (reparación, venta, ortopedia)
- Carrito de compras
- Gestión de pedidos
- Actualización semanal de inventario desde base de datos
